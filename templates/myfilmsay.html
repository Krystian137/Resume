More about the project

Overview

Movie Reviewers is a full-stack web application inspired by platforms like TMDB and Filmweb, created as part of a personal development initiative to explor web technologies. The platform allows users to browse a curated selection of movies, with interactive poster cards that reveal detailed information on hover and a dynamic carousel showcasing random featured films.

Each movie has a dedicated page containing key details, ratings, and a comment section, where registered users can engage in discussion, reply to other comments, and vote on their usefulness. The project implements a role-based access control system with three roles: admin, moderator , and standard user. Admins and moderators can add, edit, or delete movies , manage comments, moderate user profiles, and assign roles within the platform.

All movie data is fetched from the TMDB API to ensure accurate information. The platform integrates authentication, user management, form handling, and real-time feedback features to deliver a responsive and user-friendly experience. The project serves both as a movie review system and a learning ground for full-stack development practices.

Backend: Python (Flask)
Frontend: HTML5, CSS3, JavaScript, Bootstrap
Database: SQLAlchemy, Alembic
Authentication: Flask-Login, Flask-WTF
Forms & Input: WTForms, Flask-WTF, Flask-CKEditor
Data: TMDB API

Home

The homepage serves as the main entry point to the platform, offering an engaging and dynamic presentation of movie content. It features a carousel that displays three randomly selected featured films, each enriched with key information fetched from the TMDB API. Below the carousel, users are presented with a grid of interactive movie cards, which utilize CSS transitions to flip on hover, revealing details like title, release year, and rating. Additionally, movies can be sorted by various parameters like rating or release year, allowing for easy browsing.

The layout is built with Bootstrap for responsiveness, while JavaScript dynamically manages the content rendering and API data integration to ensure smooth and modern user interaction. A special feature visible only to moderators and admins is the add movie button, enabling them to add new films to the platform with ease.

Movie Site

Provides a comprehensive view of a selected film. It features the movie poster, alongside a progress circle that fills up according to the movie's rating, the percentage value is displayed at the center of the circle, movie details such as the title, release year, genre, director, writer, and overview retrieved from the TMDB API and stored in the database. Additionally, if the user is an admin or moderator, they are given the option to edit the movie details.

Comment Section

Allows users to engage with others by leaving feedback on the movie. Each comment displays the author's rating and the date and time it was posted. Additionally, users can see the upvotes and downvotes each comment has received from other logged-in users. Registered users can post comments and rate them based on their experience, giving a numerical score that reflects their opinion. Users are also able to respond to other comments, creating a threaded discussion that allows for deeper engagement and conversation. This encourages active participation and the sharing of thoughts on the movie.

The author of a comment has the ability to edit or delete their own comment at any time, providing control over their contributions. However, admins and moderators have the privilege to delete any comment, regardless of its author, to maintain the integrity of the platform and ensure the conversation stays respectful.

Add & Edit Movie

This section is dedicated to adding and editing movies in the database and is accessible only to users with admin or moderator privileges.

The process starts on the Add Movie page, where an authorized user can enter a film title into the search bar. A list of matching and similar titles is displayed, allowing the user to select the correct movie from the results.

Upon selection, the user is redirected to a shared interface that also serves as the Edit Movie page. Here, a form is pre-filled with all relevant movie data, fetched dynamically from the TMDB API — including title, release year, genres, director, writer, poster, rating, and overview.

Before final submission, the user has the opportunity to review and correct the information to ensure accuracy. Once confirmed, the movie is added to the platform and becomes accessible to all users on the main page and in search results.

User Profile

Admins and moderators have access to user profile pages in order to monitor user activity within the comment sections. This includes reviewing all comments posted by a user and assessing their behavior on the platform. If necessary, they can remove any inappropriate or disruptive comments directly from the user’s profile view. This functionality ensures a safe and respectful environment for all users and gives moderators the tools needed to enforce community standards.

Admin and moderator

Admins and moderators have access to a dedicated user management page, where they can view a complete list of registered accounts. From this panel, they can promote or demote user roles, remove users from the platform, and access individual user profiles to monitor activity or take necessary moderation actions. This feature provides full control over user administration, ensuring that only trusted members hold higher privileges and that the platform remains well-moderated.

About me and contact

The About Me page offers an overview of the Movie Reviewers. This section is optimized as a SEO-friendly subpage to enhance discoverability and answer common user questions, such as how the site works, where data is sourced from, and who manages content and moderation.

The Contact Me page provides a direct line of communication for users and potential collaborators to reach out to the site administrators. Whether it's for reporting bugs, sharing feedback, business inquiries, or collaboration proposals, this form allows messages to be sent securely and efficiently via email. It reinforces transparency and keeps the community engaged and connected with the development team.

Both sections are designed to strengthen user trust, provide transparency, and foster community interaction around the project.

Login/Register

The platform includes a secure login and registration system that allows users to create personalized accounts. Upon registration, user passwords are safely hashed before being stored in the database, ensuring data protection and user privacy.

Once logged in, users unlock full access to interactive features such as writing and replying to comments, rating movies, and voting on the usefulness of other users opinions. This authentication layer helps maintain the quality of content and fosters meaningful discussions by holding users accountable for their activity.

The system lays the groundwork for role-based access control, ensuring that only authenticated users can engage with content and that higher privileges are granted only to moderators and administrators.
