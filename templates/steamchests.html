{{ _('Idea that started the project') }}

{{ _('My friends and I used to play a lot of Counter-Strike 2, and during that time we purchased various in-game cases, which could later be sold for profit. I wanted to create a platform where we could track the value of our inventories, monitor price changes over time, and determine the best moment to sell. This project also served as a valuable opportunity for me to improve my development skills and learn new technologies.') }}

{{ _('More about the project') }}

Overview

{{ _('This web application was developed using the ') }}{{ _('Flask') }} {{ _('framework and provides tools for tracking the prices of CS2 in-game cases listed on the Steam market. Users can view historical price charts, monitor daily, weekly, monthly, and total value changes, and create accounts to manage their personal inventories. The system automatically calculates the total value of owned cases and visualizes market trends to help identify optimal selling moments.') }}

{{ _('Technologies Used') }}

Backend: Python (Flask)
{{ _('Database:') }} SQLAlchemy
Frontend: HTML5, CSS3, JavaScript, Bootstrap
{{ _('Authentication:') }} Flask-Login, Flask-WTF
{{ _('Charts:') }} Chart.js
{{ _('Data:') }} Steam Marketplace API, JSON

{{ _('Home') }}

{{ _('The homepage displays a dynamically generated list of all Counter-Strike 2 cases available on the Steam Market, each represented by a card containing the case name, image fetched from the Steam API, and market price.') }} {{ _('Clicking on a case redirects the user to a detailed subpage showing a historical price chart and metadata.') }} {{ _('Data is regularly fetched and stored in JSON format as part of my learning process with JSON handling.') }} {{ _('To improve performance, I created a separate JSON file that stores only the latest prices for each case.') }}

{{ _('User Chests') }}

{{ _('The user dashboard enables authenticated users to input and manage their owned Counter-Strike 2 cases along with quantities.') }} {{ _('Each entry is displayed in a dynamic, responsive table that includes the case’s latest market price, total value (with and without Steam transaction fees), and percentage change from the previous day.') }} {{ _('Users can update or remove individual case records.') }} {{ _('Below the table, the application calculates and displays the cumulative value of all listed cases, both pre- and post-tax.') }}

{{ _('Chest Page') }}

{{ _('The chest detail page presents an interactive price history chart powered by Chart.js, allowing users to analyze case price fluctuations over various time ranges, including the past week, month, year, and all-time.') }} {{ _('Users can dynamically select the desired time frame.') }} {{ _('Below the chart, a data table displays historical prices by date along with the corresponding percentage change compared to the previous recorded value.') }}

{{ _('Calculator') }}

{{ _('The calculator is designed for users who prefer not to create an account.') }} {{ _('It offers similar functionality to the user inventory page, allowing users to input the cases they own and their respective quantities.') }} {{ _('However, it does not persist any data — once the session ends or the page is refreshed, all information is cleared.') }} {{ _('This tool provides a quick way to estimate the total value of owned cases, both before and after Steam transaction fees.') }}
