I created these projects as a way to learn through practice. Each project helped me grow both technically and creatively, and gave me a better understanding of how to connect different technologies into complete, functional solutions. Snake Game (Flask)

A modern take on the classic Snake game – built with Flask and JavaScript. The game features multiple difficulty levels, a nickname-based scoring system, and local score saving using a JSON-based database. All gameplay logic runs entirely in the browser, providing a smooth and responsive experience. Players can compete for a spot in the top 10 leaderboard. The project also demonstrates full integration between the frontend and a lightweight Python backend.

US States Game

An interactive geographical quiz built with Python and the Turtle graphics library. The goal is to correctly name all 50 U.S. states, which are then dynamically labeled on a blank map. The application loads and processes a CSV file containing state names and their coordinates, and uses Pandas to manage data input, filtering, and output. If the player chooses to exit early, the program automatically generates a new CSV file — states_to_learn.csv — listing the states that were missed, allowing for personalized review and improvement. As a light introduction to data-driven applications, this project demonstrates how basic data science tools can be applied in an interactive and visual context.

Pong Game

A modern take on the classic Pong game, developed using Python and the Turtle graphics library. The project is structured in an object-oriented way, with separate classes for the paddles, ball, and scoreboard. It features real-time input handling for smooth two-player gameplay, dynamic collision detection, score tracking, and a gradually increasing ball speed. I built this project as a personal exercise to better understand object-oriented programming and basic game mechanics, while experimenting with simple animation and user interaction in Python.

Snake Game

A classic Snake game built in Python using the Turtle graphics library. The project is structured in an object-oriented manner, with separate classes handling the snake's behavior, food spawning, and scorekeeping. It features real-time input handling, collision detection with walls, food, and the snake’s own body, as well as persistent high score tracking using a local file. I created this project to strengthen my understanding of object-oriented programming, collision logic, and working with persistent data.

Turtle Crossing Game

A Frogger-style game developed in Python with the Turtle graphics module. The player controls a turtle trying to cross a busy road without getting hit by randomly generated cars. Each successful crossing increases the game level and the speed of oncoming traffic, making the game progressively more challenging. This project helped me improve my skills in event-driven programming, object-oriented design, and handling dynamic difficulty scaling.

Top 100 Songs

Built a Python application that scrapes the Billboard Hot 100 chart for a user-specified date (using BeautifulSoup), searches for each track on Spotify via the Spotipy API, and automatically creates a private Spotify playlist with the found songs. Utilized OAuth for user authentication and integrated data from multiple external sources (Billboard and Spotify APIs).
