Idea that started the project

I wanted to create a project that would help me develop more practical programming skills. Initially, the idea was based on concepts I had learned during various courses, and the website was intended to be a small learning exercise. However, over time, it evolved into a more advanced platform as I continued to use it as a workspace for experimenting with new technologies.

More about the project

Overview

Movie Reviewers is a full-stack web application inspired by platforms like TMDB and Filmweb, created as part of a personal development initiative to explore and master modern web technologies. The platform allows users to browse a curated selection of movies, with interactive poster cards that reveal detailed information on hover and a dynamic carousel showcasing random featured films.

Each movie has a dedicated page containing key details, ratings, and a comment section, where registered users can engage in discussion, reply to other comments, and vote on their usefulness. The project implements a role-based access control system with three levels: admin, moderator , and standard user. Admins and moderators can add, edit, or delete movies , manage comments, moderate user profiles, and assign roles within the platform.

All movie data is dynamically fetched from the TMDB API to ensure up-to-date and accurate information. The platform integrates authentication, user management, form handling, and real-time feedback features to deliver a responsive and user-friendly experience. The project serves both as a robust movie review system and a learning ground for full-stack development practices.

Technologies Used

Backend: Python (Flask)
Frontend: HTML5, CSS3, JavaScript, Bootstrap
Database: SQLAlchemy, Alembic
Authentication: Flask-Login, Flask-WTF

Data: TMDB API

Home

The homepage serves as the main entry point to the platform, offering an engaging and dynamic presentation of movie content. It features a carousel that displays three randomly selected featured films, each enriched with key information fetched in real-time from the TMDB API. Below the carousel, users are presented with a grid of interactive movie cards, which utilize CSS transitions to flip on hover, revealing details like title, release year, and rating. Additionally, movies can be sorted by various parameters like rating or release year, allowing for easy browsing.

The layout is built with Bootstrap for responsiveness, while JavaScript dynamically manages the content rendering and API data integration to ensure smooth and modern user interaction. A special feature visible only to moderators and admins is the add movie button, enabling them to add new films to the platform with ease.

Movie Site

Provides a comprehensive view of a selected film. It features the movie poster, alongside a progress circle that fills up according to the movie's rating, the percentage value is displayed at the center of the circle. Essential movie details such as the title, release year, genre, director, writer, and all overview retrieved from the TMDB API and stored in the database. Additionally, if the user is an admin or moderator, they are given the option to edit the movie details.
