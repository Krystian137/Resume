Idea that started the project

I wanted to create a project that would help me develop more practical programming skills. Initially, the idea was based on concepts I had learned during various courses, and the website was intended to be a small learning exercise. However, over time, it evolved into a more advanced platform as I continued to use it as a workspace for experimenting with new technologies.

More about the project

Overview

This web application was developed using the Flask framework and provides tools for tracking the prices of CS2 in-game cases listed on the Steam market. Users can view historical price charts, monitor daily, weekly, monthly, and total value changes, and create accounts to manage their personal inventories. The system automatically calculates the total value of owned cases and visualizes market trends to help identify optimal selling moments.

Technologies Used

Backend: Python (Flask)
Database: SQLAlchemy
Frontend: HTML5, CSS3, JavaScript, Bootstrap
Charts: Chart.js
Data: Steam Marketplace API, JSON
